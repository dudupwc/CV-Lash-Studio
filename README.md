# 💅 CV Lash Studio — Sistema de Agendamento

Sistema profissional de agendamento online para estúdio de cílios, com painel administrativo, controle inteligente de horários e envio automático de e-mails.

## 🚀 Funcionalidades

- Agendamento online de serviços
- Controle de horários com base na duração
- Bloqueio automático de conflitos
- Painel administrativo completo
- Integração com Supabase (banco de dados)
- Autenticação de administrador
- Envio automático de e-mails (EmailJS)
- Interface responsiva (funciona em celular)

## 🛠 Tecnologias

- HTML, CSS, JavaScript
- Supabase (Database + Auth)
- EmailJS (envio de e-mails)

## 📱 Acesso

Sistema desenvolvido para uso real em estúdios de beleza.
